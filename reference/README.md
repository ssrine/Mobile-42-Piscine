# React Native Expo Reference Project

This is a reference/template project for 42 Mobile Piscine exercises.

## Features:
- ✅ Expo 54.0.0
- ✅ React 19.1.0
- ✅ React Native 0.81.5
- ✅ Works with iPhone Expo Go app (SDK 54)
- ✅ Minimal setup (no extra configs)

## How to use:
1. Copy this directory for each new exercise:
   ```bash
   cp -r ex00 ex01
   cd ex01
   npm install
